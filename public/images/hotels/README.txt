Image Dimensions Guide:

1. Hero Images (hero.jpg):
   - Size: 1920x1080 pixels
   - Purpose: Main banner image for hotel page
   - Location: /[hotel-slug]/hero.jpg

2. Gallery Images (gallery-X.jpg):
   - Size: 800x600 pixels
   - Purpose: Hotel gallery images
   - Location: /[hotel-slug]/gallery-1.jpg, gallery-2.jpg, etc.

3. Room Images (rooms/[room-type].jpg):
   - Size: 600x400 pixels
   - Purpose: Individual room type images
   - Location: /[hotel-slug]/rooms/[room-type].jpg

Replace the .txt files with actual .jpg images using these dimensions.
