Image Dimensions Guide:

1. Hero Images (hero.jpg):
   - Size: 1920x1080 pixels
   - Purpose: Main banner image for activity page
   - Location: /[activity-slug]/hero.jpg

2. Gallery Images (gallery-X.jpg):
   - Size: 800x600 pixels
   - Purpose: Activity gallery images
   - Location: /[activity-slug]/gallery-1.jpg, gallery-2.jpg, etc.

3. Feature Images (features/[feature-name].jpg):
   - Size: 600x400 pixels
   - Purpose: Images for specific features or highlights
   - Location: /[activity-slug]/features/[feature-name].jpg

Replace the .txt files with actual .jpg images using these dimensions.
