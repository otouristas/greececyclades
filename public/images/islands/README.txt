Image Dimensions Guide:

1. Hero Images (hero.jpg):
   - Size: 1920x1080 pixels
   - Purpose: Main banner image for island page
   - Location: /[island-slug]/hero.jpg

2. Gallery Images (gallery-X.jpg):
   - Size: 800x600 pixels
   - Purpose: Island gallery images
   - Location: /[island-slug]/gallery-1.jpg, gallery-2.jpg, etc.

3. Feature Images (features/[feature-name].jpg):
   - Size: 600x400 pixels
   - Purpose: Images for specific attractions or features
   - Location: /[island-slug]/features/[feature-name].jpg

4. Category Images:
   - Beaches (beaches/[beach-name].jpg): 800x600 pixels
   - Villages (villages/[village-name].jpg): 800x600 pixels
   - Landmarks (landmarks/[landmark-name].jpg): 800x600 pixels
   - Culture (culture/[culture-name].jpg): 800x600 pixels

Replace the .txt files with actual .jpg images using these dimensions.
